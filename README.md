### IceGrain
[![][travis-badge]][travis-link]

A light-weight SNS.

#### External Dependencies

+ Node.js
+ npm
+ MongoDB

#### How to build

Make sure that you have installed **npm** and **mongodb**.

1. `git clone git://github.com/sakurapleasure/IceGrain.git`
2. `cd IceGrain`
3. `npm install`

#### How to launch
`node index.js`
